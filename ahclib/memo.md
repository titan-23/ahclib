# ビジュアライザ生成の指示

AHC の問題文を渡す。競技者がローカルで使う HTML ビジュアライザを作成し、ファイルに書き出す。

## 出力

- 生成物はカレントディレクトリの `visualizer.html` に直接書き出す。コードブロックでの貼り付けは不要。
- `ahclib` の `vis` ダッシュボードがこのファイルを読み、ビジュアライザタブに iframe で埋め込む。

## 構成

- HTML、CSS、JavaScript を1つの HTML ファイルにまとめる。
- 外部ライブラリや CDN は使わない。バニラ JS で実装する。

## データ入力

- 入力データと出力データは、HTML の読み込み前にグローバル定数として定義済みとする。`INPUT_DATA` と `OUTPUT_DATA` の2つで、いずれも文字列型。
- ダッシュボードは `</body>` の直前に、`INPUT_DATA` と `OUTPUT_DATA` を定義する `<script>` を差し込む。
- `onload` でこれらの定数をパースし、描画を開始する。

## 機能

- 描画エリアは Canvas を使い、盤面の状態を表示する。
- ターン制の問題では、シークバーで任意のターンへ移動できるようにする。再生・一時停止ボタンと再生速度の変更を付ける。
- 現在のターンのスコアを表示する。
- 盤面のサイズや、カーソルを合わせたセルの詳細を表示する領域を設ける。表形式の情報を加えてもよい。

## デザイン

- 長時間見ても疲れない簡素なデザインにする。ダークモードに合うと望ましいが必須ではない。
